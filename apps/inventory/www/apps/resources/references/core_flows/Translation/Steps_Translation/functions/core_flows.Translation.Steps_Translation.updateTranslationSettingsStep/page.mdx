# updateTranslationSettingsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `updateTranslationSettingsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step updates translation settings based on the provided input.
It supports both single and multiple translation settings updates.
In case of failure, it compensates by restoring the previous translation settings.

:::note

This is available starting from [Medusa v2.13.0](https://github.com/medusajs/medusa/releases/tag/v2.13.0).

:::

## Example

```ts
const result = updateTranslationSettingsStep({
  id: "ts_123",
  fields: ["title", "description", "material"],
  is_active: false
})
```
