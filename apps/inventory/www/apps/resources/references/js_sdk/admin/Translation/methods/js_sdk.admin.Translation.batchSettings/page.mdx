# batchSettings

This method allows bulk operations on translation settings. It sends a request to the
[Batch Translation Settings](https://docs.medusajs.com/api/admin#translations\_posttranslationssettingsbatch)
API route.

:::note

This is available starting from [Medusa v2.12.6](https://github.com/medusajs/medusa/releases/tag/v2.12.6).

:::

## Example

```ts
sdk.admin.translation.batchSettings({
  create: [
    {
      entity_type: "product",
      fields: ["title", "description"],
      is_active: true
    }
  ],
  update: [
    {
      id: "trset_123",
      fields: ["title", "description", "subtitle"],
      is_active: true
    }
  ],
  delete: ["trset_456"]
})
.then(({ created, updated, deleted }) => {
  console.log(created, updated, deleted)
})
```
```
