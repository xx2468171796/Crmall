# {metadata.title}

<Note>

The `codemod` command is available from [Medusa v2.13.0](https://github.com/medusajs/medusa/releases/tag/v2.13.0).

</Note>

Run a codemod to update your Medusa project to the latest version.

Learn more about codemods and available transformations in the [Codemods](!docs!/learn/codemods) guide.

```bash npx2yarn
npx medusa codemod <transformation>
```
