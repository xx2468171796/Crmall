# deleteTranslationSettingsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `deleteTranslationSettingsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step deletes translation settings based on the provided IDs.
It compensates by restoring the deleted translation settings in case of failure.

:::note

This is available starting from [Medusa v2.13.0](https://github.com/medusajs/medusa/releases/tag/v2.13.0).

:::

## Example

```ts
const result = deleteTranslationSettingsStep([
  "ts_123",
  "ts_456"
])
```
